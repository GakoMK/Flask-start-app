# Flask starter

Used in the project:<br/>
- flask-admin
- flask-migrate
- flask_sqlalchemy
<br/>
<br/>
**This project was created while learning Flask, maybe it will be useful to someone.**<br/><br/>


## How to start:
1. Create folder 'data'
2. Python:
```
manage.py db init
manage.py db migrate
manage.py db upgrade
main.py
```
